Detailed Description

This class implements Spectral Sparsifier by Effective Weight Sampling. Original paper https://arxiv.org/pdf/0803.0929.pdf

Constructor to create a ISparsifier object to perform Spectral Sparsification by Effective Resistance

Parameters
graphSmart Pointer to the IGraph object. This is the dense graph for the sparsifier.
CC hyper-parameter of Spectral Sparsifier by Effective Resistance. As described by paper. This should be a large constant. Default value is 4.0.
EpsilonEpsilon hyper-parameter. This should be between 0.0 and 1.0 The sparsifier will generate (m log m ) / (Epsilon^2) edges. Thus, lower epsilon will yeild denser sparsifier. High epsilon could yeild a disconnect graph.
ERPolicyEffectiveResistance calculation method. APPROXIMATE_ER: Default. Approximate EffectiveResistance with an error. This should be sufficient for Spectral Sparsifier.
EXACT_ER: Calculate graph exact EffectiveResistance. This is very slow and not recommended.
