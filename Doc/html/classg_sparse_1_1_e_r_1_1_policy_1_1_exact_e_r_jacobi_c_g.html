Detailed Description

This class implements Spectral Sparsifier by Effective Weight Sampling. Adaptation from http://ccom.uprrp.edu/~ikoutis/SpectralAlgorithms.htm The algorithm leverages Conjugated Graident with Jacobi preconditioner to solve linear system
